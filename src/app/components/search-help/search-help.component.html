Syntaxe de la recherche

Par défaut, Pibui recherche les titres/artistes ayant au moins un des mots cités.
Mais différents opérateurs permettent d'affiner la recherche :

  • + : Le mot qui suit doit obligatoirement figurer dans touts les résultats obtenus.
  • - : Le mot qui suit ne peut apparaître dans aucun des résultats retournés.
  • * : Le caractère de remplacement, qui indique zéro ou plusieurs caractères. Il ne peut apparaître qu'à la fin d'un mot.
  • " : Tout ce qui est entre guillemets est considéré comme un tout.

Donc, par exemple :

  • Pour trouver les titres/artistes qui contiennent Good et/ou Morning : Good Morning
  • Pour trouver les titres/artistes qui contiennent Good mais pas Morning : +Good -Morning
  • Pour trouver les titres/artistes qui contiennent exactement Good Morning : "Good Morning"
  • Pour trouver les titres/artistes dont un mot commence par Morn : Morn*
